# SDET Mastery Framework

Professional QA Automation framework built with **Java** and **Selenium WebDriver**.

## Architecture
This project implements the **Page Object Model (POM)** design pattern, focusing on:
* **Inheritance:** Using a `BasePage` to centralize WebDriver configurations.
* **Encapsulation:** Protecting locators with private access modifiers and public getters.
* **Synchronization:** Implementing Explicit Waits (`WebDriverWait`) for robust execution.
* **Documentation:** Fully documented using professional Javadocs.

## Tech Stack
* **Java 22**
* **Selenium 4.x**
* **Maven**
* **WebDriverManager**
